## Why Use This API?

Our API provides everything you need to launch an e-commerce platform with ease. Whether you're building a SaaS service that supports multiple shops, or just need a reliable and secure solution for individual stores, this API covers all essential features.

<CardGroup cols={2}>
  <Card
    title="Authentication"
    icon="lock"
    href="https://mintlify.com/docs/settings/global"
  >
      All requests require user authentication. Authentication is handled by Clerk and ensures security and privacy for both store owners and customers.
  </Card>
  <Card
    title="Store Management"
    icon="store"
    href="https://mintlify.com/docs/api-playground/openapi"

      Users can create, update, and manage their stores, including settings for products, payments, and more.
  </Card>
  <Card

    icon="credit-card"

  </Card>
  <Card
    title="Customization"
    icon="brush"

  >
      Tailor your store with various customization options, from logos and colors to category layouts.
  </Card>
</CardGroup>
